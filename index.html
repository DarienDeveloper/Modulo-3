TEMA 1-TEXTO

Propiedad Valor Inicial Descripción
color Depende del navegador Define el color del texto.
font-family Depende del navegador Especifica la fuente o grupo de fuentes para el texto.
font-size medium Establece el tamaño del texto.
font-weight normal Controla el grosor del texto (por ejemplo: normal, bold).
font-style normal Define el estilo del texto (normal, italic, oblique).
font-variant normal Controla la representación de las mayúsculas (por ejemplo, small-caps).
font Depende del navegador Agrupa varias propiedades de la fuente (font-size, font-family, etc.).
text-align start (depende de la dirección del texto) Alinea el texto dentro de su contenedor (left, center, right).
line-height normal Define la altura de la línea (distancia entre líneas de texto).
text-decoration none Aplica decoraciones al texto (subrayado, tachado, etc.).
text-transform none Controla la capitalización del texto (uppercase, lowercase).
vertical-align baseline Alinea elementos en línea en relación con la línea base del texto.
text-indent 0 Indenta la primera línea de un bloque de texto.
letter-spacing normal Define el espacio entre caracteres del texto.
word-spacing normal Define el espacio entre palabras.
white-space normal Controla cómo se manejan los espacios en blanco en el texto.

TEMA 2 - ENLACES

Característica Propiedades Descripción
Tamaño, Color y Decoración font-size, color, text-decoration Controla el tamaño, el color y la decoración de los enlaces, como subrayado.
Decoración Personalizada text-decoration, background-color Permite aplicar decoraciones personalizadas, como subrayado de colores o un fondo detrás del enlace.

TEMA 3 - IMÁGENES

Característica Propiedades Descripción
Anchura y Altura width, height Permite definir las dimensiones de una imagen.
Eliminar el Borde border: none Elimina el borde que puede aparecer alrededor de una imagen.

TEMA 4 - LISTAS

Propiedad Valor Inicial Descripción
list-style-type disc Define el tipo de marcador (círculo, cuadrado, número, etc.) que se usa para los elementos de la lista.
list-style-position outside Controla si el marcador aparece dentro o fuera del cuadro de contenido.
list-style-image none Permite usar una imagen como marcador de lista.
list-style Depende del navegador Es un atajo que combina varias propiedades de estilo de lista (list-style-type, list-style-position, etc.).

TEMA 5 - TABLAS

Propiedad Se Aplica a Valor Inicial Descripción
border-collapse Tablas separate Define si los bordes de la tabla están separados o colapsados en un único borde.
border-spacing Tablas 0 Establece el espacio entre los bordes de las celdas cuando se usa border-collapse: separate.
empty-cells Celdas de tabla show Define si se muestran los bordes de las celdas vacías o no.
caption-side Tablas top Controla la posición de la leyenda de la tabla (arriba o abajo).
